## ?? Conseils pour Apprendre

### 1. Ne Pas Juste Lire, Coder
```
Lire ? ? Comprendre d'abord
Lire ? Coder ? Tester ? Comprendre ?
```

### 2. Commencer Simple
```
? Refactoring complet de 1000 lignes
? Renommer une méthode
? Puis chercher 5 méthodes
? Puis les renommer toutes
? Puis ajouter du logging
```

### 3. Valider Avant de Transformer
```csharp
// D'abord: vérifier que ça existe
var methods = context.FindMethods().WithName("X").Execute();
if (!methods.Any()) return; // Pas trouvé!

// Ensuite: transformer
var editor = CodeEditor.FromCode(code);
editor.RenameMethod("X", "Y").Apply();
```

### 4. Garder les Originaux
```csharp
var originalCode = code;
var editor = CodeEditor.FromCode(code);

// ...opérations...

var result = editor.Apply();
if (!result.Success)
{
    // Utiliser originalCode pour recommencer
}
```
