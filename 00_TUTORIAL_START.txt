## ?? Conseils pour Apprendre

### 1. Lire ? Comprendre ? Coder
```
Lire l'exemple
    ?
Comprendre ce qu'il fait
    ?
Écrire le code vous-même
    ?
Tester et modifier
    ?
Progresser! ??
```

### 2. Commencer Simple
- Chercher une méthode
- Puis ajouter un filtre
- Puis transformer
- Puis chaîner plusieurs transformations

### 3. Tester Itérativement
```csharp
// D'abord, chercher
var methods = context.FindMethods().WithName("X").Execute();
Console.WriteLine($"Trouvé: {methods.Count()}");

// Ensuite, transformer
var result = editor.RenameMethod("X", "Y").Apply();
Console.WriteLine($"Succès: {result.Success}");

// Vérifier
Console.WriteLine(result.ModifiedCode);
```
