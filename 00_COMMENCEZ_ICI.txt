## 🎓 Exemple Real-World

```csharp
// Chercher toutes les méthodes "Get" publiques
var context = CodeContext.FromCode(code);
var methods = context.FindMethods()
    .WithNameContaining("Get")
    .IsPublic()
    .Execute()
    .ToList();

// Les renommer en "Fetch"
var editor = CodeEditor.FromCode(code);
foreach (var method in methods)
{
    var newName = method.Identifier.Text.Replace("Get", "Fetch");
    editor.RenameMethod(method.Identifier.Text, newName);
}

// Appliquer et sauvegarder
var result = editor.Apply();
editor.SaveToFile("refactored.cs");
```
