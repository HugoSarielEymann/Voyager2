## ?? INTÉGRATION

S'intègre parfaitement avec CodeSearcher:

```csharp
// Chercher les anciennes propriétés
var context = CodeContext.FromCode(code);
var oldProps = context.FindVariables()
    .WithNameContaining("Old")
    .Execute();

// Mapper vers nouvelles
var engine = new PropertyMappingEngine();
foreach (var prop in oldProps)
    engine.AddMapping(prop.Identifier.Text, "New.Path");

// Transformer
var result = engine.TransformCode(code);

// Optionnellement, refactoring additionnel
var editor = CodeEditor.FromCode(result.TransformedCode);
// ... plus de transformations
```
