### Chercher (Core)
```csharp

    .IsPublic()
    .IsAsync()
    .ReturningTask()
    .Execute()
```

### Transformer (Editor)
```csharp
editor.RenameMethod("old", "new")
    .WrapWithTryCatch("method", "handler")
    .Replace("x", "y")
    .Apply()
```

### Ensemble (Intégration)
```csharp
// Trouver ? Transformer ? Appliquer
var methods = context.FindMethods().WithName("Get*").Execute();
var editor = CodeEditor.FromCode(code);
foreach (var m in methods) editor.RenameMethod(m.Identifier.Text, "Fetch*");
var result = editor.Apply();
```
